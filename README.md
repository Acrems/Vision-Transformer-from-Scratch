# Vision-Transformer-from-Scratch
A from-scratch PyTorch implementation of the Vision Transformer (ViT) architecture, trained on the CIFAR-10 dataset as part of winter project conducted by Electrical Engineering Association at IIT Kanpur.It explores how Transformers can be applied to computer vision tasks by treating image patches as sequences, bypassing traditional convolutional operations. The model is trained and evaluated on the CIFAR-10 dataset.
